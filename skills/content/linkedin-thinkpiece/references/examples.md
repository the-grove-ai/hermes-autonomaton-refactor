# Structural Examples

Annotations of published pieces illustrating structural moves. For calibration, not reproduction.

## Example 1: News-Anchored Analysis

**Pattern:** "Sam Altman Just Warned You" style — specific event → hidden implications

```
HOOK: Specific event (Altman's developer conference announcement)
      → pivots within 2 paragraphs to "what didn't make headlines"

BUILD 1: "Sign in with ChatGPT" as identity layer play
      → specific mechanism (OAuth flow), comparison (Google's playbook)

BUILD 2: Memory portability risk
      → direct address ("Think about the last six months...")
      → makes abstract risk personal

BUILD 3: Enterprise shadow IT at cognitive layer
      → stakeholder expansion (individual → enterprise)

BUILD 4: Infrastructure lock-in dynamics
      → historical pattern (remembers when this happened before)
      → structural analysis (why this time is different)

CASCADE: "For individuals... For businesses... For builders..."
      → each gets a specific implication, not a generic warning

CLOSE: Window-is-closing urgency, no CTA
```

## Example 2: Research Translation

**Pattern:** "The Oracle Trap" style — counterintuitive finding → named mechanism

```
HOOK: Specific research (Kevin Munger essay, Anthropic study)
      → leads with counterintuitive finding (AI making us worse, not better)

BUILD: Constructs "cognitive debt" as a concept
      → specific data (17% skill loss from Anthropic study)
      → names the mechanism, not just the outcome

CLOSE: Urgency framed as institutional risk, not personal failing
```

## Example 3: Governance Reframe

**Pattern:** "Where Your AI Lives" style — buried lede in document → civilizational question

```
HOOK: Specific document (Anthropic's revised Claude constitution)
      → the "buried lede" move (what document actually says vs. coverage)

BUILD: Escalates from technical observation to civilizational question
      → Stanford agent simulation as evidence of emergence
      → "where your AI lives" as infrastructure question hiding inside ethics question

CLOSE: The choice being made by default, without conscious decision
```

## Example 4: Competitive Analysis with Structural Argument

**Pattern:** "The Most Expensive Ad-Free Ad in History" style — company move → industry physics → historical parallel

```
HOOK: Specific event (Anthropic $8M Super Bowl ad) + 1984 Apple parallel
      → pivots to "distraction from a more dangerous question"

BUILD 1: "The CEO Doth Protest Too Much"
      → Altman's defensive response as evidence of position shift
      → documented timeline (20-month journey from "last resort" to "$60 CPM")

BUILD 2: "The Thermodynamics of Concentration"
      → specific financials (800M users, 35M paying, $7B losses)
      → "This isn't a moral argument. It's physics."

BUILD 3: Historical compression argument
      → PC revolution had decades to distribute; AI compressed to 18 months
      → open-source exists but concrete being poured anyway

BUILD 4: Structural metaphor (Prussian Waldsterben)
      → not decorative — parallel is structural (monoculture strips soil)
      → "displaced people become customers, not competitors"

CLOSE: Choice framing — "plant a forest, or keep arguing about which company owns the rows"
      → no CTA, no product, urgency without panic
```
