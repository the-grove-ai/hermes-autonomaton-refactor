# Expense Tracking & Personal Finance — Goal Context

## Purpose

Track expenses consistently with clean separation between Grove
Foundation expenses, Take Flight Advisors expenses, personal expenses,
and the carriage house construction project. This separation matters
for tax prep and grant reporting.

## Categories

### grove_foundation
Standards work, hosting (Vercel, domains), legal (Dentons — trademark
filings), travel for Grove business, conference fees, API costs for
Hermes/Λ Watch development, design/branding expenses.

### take_flight
Consulting practice expenses. Client-related travel, tools, any
DrumWave/Monarch/other client engagement costs.

### personal
Non-business expenses.

### carriage_house
Construction project expenses. Cross-referenced with the Notion
construction budget database for detailed tracking. This category
exists here for aggregate financial visibility.

## Tracking Approach

Keep it low-friction. Mylo should be able to log an expense from a
quick description:

"$45 at Lowe's for the garage" → carriage_house, $45, Lowe's, today
"Vercel bill $20" → grove_foundation, $20, Vercel, today
"Dentons invoice $2,500" → grove_foundation, $2,500, Dentons, today

Monthly summaries should show total by category and flag any large
or unusual items.

## Tax Considerations

Grove Foundation is a 501(c)(6). Foundation expenses need to be
tracked separately for any grant reporting requirements. Keep receipts
and invoices — the Notion construction budget database has an Invoice
File field for the carriage house; consider a similar approach for
other categories if volume warrants it.
